# Basic documentation
## Commands:
- /sudo player message/command - Send a message or run command as another player!
### Examples:
- /sudo ErikX Hi There!
- /sudo ErikX /say HI there!

## Permissions:

| Permission    | Default       | Description                  |
| ------------- |:-------------:| ----------------------------:|
| sudo.use      |      OP       | Permission to sudo a player! |
